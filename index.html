American Sign Language

American Sign Language (ASL) is a complete, natural language that has the same linguistic properties as spoken languages, with grammar that differs from English. ASL is expressed by movements of the hands and face.

Braille

Braille is a system of touch reading and writing for blind persons in which raised dots represent the letters of the alphabet. It also contains equivalents for punctuation marks and provides symbols to show letter groupings.

English

English is a spoken and written language that uses a set of symbols called letters to represent sounds. These letters are combined to form words, which in turn are combined to form sentences.
